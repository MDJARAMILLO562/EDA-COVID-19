# EDA COVID-19
Exploratory Data Analysis (EDA) realizado como parte de un ejercicio académico en la Maestría en Ciencia de Datos.

## Objetivo
Explorar, limpiar y analizar datos relacionados con el impacto de pacientes reportados por COVID-19, identificando patrones generales, valores faltantes y distribuciones relevantes.

## Metodología
- Carga y exploración inicial del dataset
- Limpieza y estructuración de datos
- Análisis descriptivo
- Visualización de resultados

## Herramientas
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook (Google Colab)

## Notas
Repositorio con fines académicos que demuestra habilidades en análisis exploratorio de datos, estructuración y gestión de información.
